# Інформаційна система для розпізнавання еталонного сигналу при невідомому коефіцієнті стискання в часі
## Опис проекту
Ця програма реалізує алгоритм розпізнавання еталонного сигналу при невідомому коефіцієнті стискання в часі, заснований на використанні функцій непропорційності. 
## Постановка задачі
Задано певний набір еталонних сигналів, кожен з яких можна представити формулою *y=f(t)*, де *y* - значення сигналу, а *t* - час
При розпізнаванні сигналу відбулось стиснення в часі, та в амплітуді, через що сигнал, який аналізується має вигляд
*y=Kf(at + A)*, Потрібно розпізнати, які еталонні сигнали присутні в сигналі, який аналізується.
