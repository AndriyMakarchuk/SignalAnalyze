# Документація до лінтеру
## Вибір

Було обрано лінтер **Cppcheck**, так як він дозволяє перевірити наявність критичних помилок, які можуть вплинути на роботу програми

## Інструкція з запуску

Потрібно завантажити програму з сайту https://cppcheck.sourceforge.io/ 
Також на цьому сайті є список плагінів для IDE. 

Далі можна викорстовувати як окремо, так і інтегрувати в IDE

## Інтеграція в Visual studio

Спершу обов'язково потрібно завантажити встановлювач для самого застосунку з сайту, вказанного вище, та встановити.
Бажано встановлювати по стандартному шляху. Тоді плагін зможе автоматично знайти програму. 
Потім потрібно встановити сам плагін. Репозиторій плагіну - https://github.com/VioletGiraffe/cppcheck-vs-addin/releases/tag/1.5
Після встановлення плагіну якщо все встановилось правильно - то в меню Tools має з'явитись опція Check current project with cppcheck
